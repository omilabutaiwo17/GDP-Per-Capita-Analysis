# GDP-Per-Capita-Analysis

📊 GDP Per Capita Analysis

This project explores global GDP per capita data using Python for data analysis and visualization.

📁 Dataset

The dataset includes:

- Country/Territory
- Region
- IMF GDP estimates
- World Bank GDP estimates
- UN GDP estimates
  
**🔧 Tools Used**

- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook
  
**📈 Analysis Performed**

- Data cleaning (removed unnecessary columns)
- Correlation analysis between GDP estimates
- Data visualization using heatmaps
  
**🔥 Key Insight**

- Strong positive correlation between World Bank and UN GDP estimates (~0.93)
- Moderate correlation between IMF and World Bank
- Weak correlation between year variables and GDP values
